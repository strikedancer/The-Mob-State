## How to Use These Prompts

### Option 1: DALL-E 3 (ChatGPT Plus / API)
1. Go to ChatGPT with DALL-E 3 access
2. Copy each prompt individually
3. Request: "Generate this image: [paste prompt]"
4. Download as PNG
5. Resize to 512x512 or 1024x1024
6. Save to `client/images/vehicles/[filename]`

### Option 2: Midjourney
1. Join Midjourney Discord
2. Use command: `/imagine [paste prompt] --ar 1:1 --v 6 --style raw`
3. Upscale the best result
4. Download and save to `client/images/vehicles/[filename]`

### Option 3: Stable Diffusion (Local/ComfyUI)
1. Use model: Realistic Vision or SDXL
2. Paste prompt in positive prompt field
3. Negative prompt: `blurry, low quality, distorted, watermark, text, logo, background objects`
4. Settings: Steps 30-50, CFG 7-8, Sampler DPM++ 2M Karras
5. Generate and save to `client/images/vehicles/[filename]`

### Option 4: Bulk Generation Script
Create a batch script to generate all images:
```python
import openai
import os

prompts = {
    "lamborghini_aventador.png": "Professional car photography of a Lamborghini Aventador SVJ...",
    # ... add all prompts
}

for filename, prompt in prompts.items():
    response = openai.Image.create(
        prompt=prompt,
        n=1,
        size="1024x1024"
    )
    # Download and save image
```

## Post-Processing
After generating all images:
1. Resize all to consistent dimensions (512x512 recommended for game UI)
2. Ensure transparent or consistent background color
3. Optimize file size with tools like TinyPNG
4. Place all files in `client/images/vehicles/`
5. Restart Flutter app to load new images
