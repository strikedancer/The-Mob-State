# Crew Building Image Prompts

## Scope
We only create HQ images. Other upgrades (cars, boats, weapons, ammo, drugs, cash storage) are backend-only and NOT shown in images.

## Copy order (recommended)
1) COPY BLOCK: Camping HQ
2) COPY BLOCK: Rural HQ
3) COPY BLOCK: City HQ
4) COPY BLOCK: Villa HQ

## Global prompt rules
- Each image shows ONLY the HQ building for that style and level.
- Game mood: modern crime/mafia management game, gritty but clean, grounded realism, cinematic lighting.
- Tone: serious, tense, organized, professional criminal enterprise.
- Same camera angle and framing for all images of the same style; only the HQ level changes.
- No overlays, no UI, no text, no icons, no watermarks.
- Consistent lighting per style set.
- High detail, realistic game art, clean edges.

## HQ level definitions (by style)

### Camping style HQ
- Base environment: campsite/gravel lot, wooden fences, pine trees, rustic props, overcast soft light.
- HQ L0: caravan on gravel, single small parking space, basic flag pole.
- HQ L1: wooden chalet, two parking spaces, small shed.
- HQ L2: larger chalet, four parking spaces, expanded shed.
- HQ L3: premium chalet, six parking spaces, landscaped campsite.

### Rural style HQ
- Base environment: farmhouse land, dirt driveways, fields, wooden fences, warm afternoon light.
- HQ L0: small farmhouse, one parking space.
- HQ L1: renovated farmhouse, two parking spaces.
- HQ L2: large farmhouse, four parking spaces.
- HQ L3: estate farmhouse, six parking spaces.

### City style HQ
- Base environment: urban lot, concrete ground, steel fencing, night lights, clean modern vibe.
- HQ L0: compact office HQ, single parking bay.
- HQ L1: mid-rise office HQ, two parking bays.
- HQ L2: modern office HQ, four bays.
- HQ L3: luxury tower HQ, six bays.

### Villa style HQ
- Base environment: luxury estate, manicured lawn, stone walls, golden hour light.
- HQ L0: modest villa, one parking space.
- HQ L1: upgraded villa, two parking spaces.
- HQ L2: large villa, four parking spaces.
- HQ L3: ultra luxury villa, six parking spaces.

## Example prompt
"Camping style HQ only, modern crime/mafia management game mood, gritty but clean, cinematic lighting. Same camera angle, high detail, realistic. HQ Level 0 (caravan on gravel, single small parking space, basic flag pole). No text, no UI, no watermark."
