# Crew Building Prompts (Raphael AI)

## Scope
Only HQ images are created. Other upgrades (cars, boats, weapons, ammo, drugs, cash storage) are backend-only and NOT shown in images.

## Global prompt rules
- Each image shows ONLY the HQ building for that style and level.
- Single image only. No collage, no grid, no split panels, no multiple frames.
- Game mood: modern crime/mafia management game, gritty but clean, grounded realism, cinematic lighting.
- Tone: serious, tense, organized, professional criminal enterprise.
- Same camera angle and framing for all images of the same style; only the HQ level changes.
- No overlays, no UI, no text, no icons, no watermarks.
- Consistent lighting per style set.
- High detail, realistic game art, clean edges.
