### Snelle frame-to-video workflow (Windows + FFmpeg)

**Input frames (verwacht):**
- `client/assets/images/prostitution/animation/recruitment_anim_frame1_approach.png`
- `client/assets/images/prostitution/animation/recruitment_anim_frame2_contact.png`
- `client/assets/images/prostitution/animation/recruitment_anim_frame3_exchange.png`
- `client/assets/images/prostitution/animation/recruitment_anim_frame4_exit.png`
- `client/assets/images/prostitution/animation/recruitment_anim_frame5_success.png`

**Input frames (VIP variant):**
- `client/assets/images/prostitution/animation/recruitment_anim_frame1_approach.png`
- `client/assets/images/prostitution/animation/recruitment_anim_frame2_contact.png`
- `client/assets/images/prostitution/animation/recruitment_anim_frame3_exchange.png`
- `client/assets/images/prostitution/animation/recruitment_anim_frame4_exit.png`
- `client/assets/images/prostitution/animation/recruitment_anim_frame5_success_vip.png`

**Output video:**
- `client/assets/videos/prostitution/recruitment_sequence_master.mp4`
- `client/assets/videos/prostitution/recruitment_sequence_vip.mp4`

**Stap 1 (eenmalig, map maken):**
```powershell
New-Item -ItemType Directory -Force "client/assets/videos/prostitution" | Out-Null
```

**Stap 2 (frames kopiëren naar numerieke volgorde voor ffmpeg):**
```powershell
Copy-Item "client/assets/images/prostitution/animation/recruitment_anim_frame1_approach.png" "client/assets/images/prostitution/animation/frame_001.png" -Force
Copy-Item "client/assets/images/prostitution/animation/recruitment_anim_frame2_contact.png" "client/assets/images/prostitution/animation/frame_002.png" -Force
Copy-Item "client/assets/images/prostitution/animation/recruitment_anim_frame3_exchange.png" "client/assets/images/prostitution/animation/frame_003.png" -Force
Copy-Item "client/assets/images/prostitution/animation/recruitment_anim_frame4_exit.png" "client/assets/images/prostitution/animation/frame_004.png" -Force
Copy-Item "client/assets/images/prostitution/animation/recruitment_anim_frame5_success.png" "client/assets/images/prostitution/animation/frame_005.png" -Force

# VIP sequence
Copy-Item "client/assets/images/prostitution/animation/recruitment_anim_frame1_approach.png" "client/assets/images/prostitution/animation/frame_001_vip.png" -Force
Copy-Item "client/assets/images/prostitution/animation/recruitment_anim_frame2_contact.png" "client/assets/images/prostitution/animation/frame_002_vip.png" -Force
Copy-Item "client/assets/images/prostitution/animation/recruitment_anim_frame3_exchange.png" "client/assets/images/prostitution/animation/frame_003_vip.png" -Force
Copy-Item "client/assets/images/prostitution/animation/recruitment_anim_frame4_exit.png" "client/assets/images/prostitution/animation/frame_004_vip.png" -Force
Copy-Item "client/assets/images/prostitution/animation/recruitment_anim_frame5_success_vip.png" "client/assets/images/prostitution/animation/frame_005_vip.png" -Force
```

**Stap 3 (één vaste ffmpeg command):**
```powershell
ffmpeg -y -framerate 1 -i "client/assets/images/prostitution/animation/frame_%03d.png" -vf "fps=30,scale=1920:1080:force_original_aspect_ratio=decrease,pad=1920:1080:(ow-iw)/2:(oh-ih)/2:color=black,format=yuv420p" -c:v libx264 -preset medium -crf 18 -movflags +faststart "client/assets/videos/prostitution/recruitment_sequence_master.mp4"

# VIP versie
ffmpeg -y -framerate 1 -start_number 1 -i "client/assets/images/prostitution/animation/frame_%03d_vip.png" -vf "fps=30,scale=1920:1080:force_original_aspect_ratio=decrease,pad=1920:1080:(ow-iw)/2:(oh-ih)/2:color=black,format=yuv420p" -c:v libx264 -preset medium -crf 18 -movflags +faststart "client/assets/videos/prostitution/recruitment_sequence_vip.mp4"
```

**Stap 4 (optioneel smooth transitions/interpolatie):**
```powershell
ffmpeg -y -framerate 1 -i "client/assets/images/prostitution/animation/frame_%03d.png" -vf "minterpolate=fps=30:mi_mode=mci:mc_mode=aobmc:vsbmc=1,scale=1920:1080:force_original_aspect_ratio=decrease,pad=1920:1080:(ow-iw)/2:(oh-ih)/2:color=black,format=yuv420p" -c:v libx264 -preset medium -crf 18 -movflags +faststart "client/assets/videos/prostitution/recruitment_sequence_master_smooth.mp4"
```

**Als ffmpeg ontbreekt:**
```powershell
winget install Gyan.FFmpeg
```
